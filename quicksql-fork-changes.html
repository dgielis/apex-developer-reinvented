New Features

9 additions
🗄Oracle 26ai Database Version Supportv1.3.0
Added 26ai as a valid value for the db setting. Includes updated grammar docs and test cases for native boolean and yn override behavior under this version.
db settinggrammar
⏪Flashback Data Archive Directivesv1.3.1
New /flashback and /fda table-level directives. Generates ALTER TABLE … FLASHBACK ARCHIVE after table creation. Optionally accepts an archive name, e.g. /flashback myarchive.
table directiveDDL
🏷Oracle 23ai Schema Annotationsv1.3.2
New {key 'value'} syntax for Oracle SQL annotations on tables, columns, and views. Supports key-value pairs, flag annotations, and multiple annotations. Emitted as ANNOTATIONS(…) clauses in DDL.
23aiannotationstablesviews
🌐Multi-lingual Translation Directivev1.3.3
New /trans column directive (also /translation). Auto-generates a shared language table, a <table>_trans table, and a <table>_resolved view using sys_context. Configurable via transcontext.
column directivei18nviews
💬DESCRIPTION Annotation → COMMENT ONv1.3.4
Using a DESCRIPTION key in an annotation auto-generates a COMMENT ON TABLE or COMMENT ON COLUMN statement. Both single-quoted and double-quoted values are supported.
annotationscomments
🔒Immutable (Append-Only) Tablesv1.3.5
New /immutable directive creates tables with NO DROP UNTIL 0 DAYS IDLE / NO DELETE UNTIL 16 DAYS AFTER INSERT and a BEFORE INSERT trigger blocking updates/deletes. Ideal for audit logs.
table directiveauditDDL
📦SODA Collection Tablesv1.3.5
New /soda directive generates a fixed-schema SODA-compatible collection table with id, created_on, last_modified, version, and json_document columns.
SODAJSONtable directive
📂Table Groups via TGROUP Annotationv1.3.5
New {TGROUP 'name'} annotation (renamed from GROUP, a reserved word) generates INSERT statements into USER_ANNOTATIONS_GROUPS$ for logical table grouping.
annotationsorganization
🔷DOMAIN Types, SDO_GEOMETRY & Duality Viewsv1.3.13
Added support for Oracle DOMAIN types, SDO_GEOMETRY spatial column type, and JSON-Relational Duality Views — bringing the library up to date with Oracle 23ai+.
23aispatialduality views

Bug Fixes

14 fixes
🏷Reserved Word Prefix Skipped with User Prefixv1.3.0
Tables named with reserved keywords (e.g. user) were always prefixed with the_. Now, if a user-defined prefix is already set, the extra the_ prefix is correctly skipped.
📅Default Date Expressions No Longer Quotedv1.3.5
SQL date expressions like sysdate, current_timestamp, systimestamp used with /default were incorrectly wrapped in quotes. Now generated verbatim.
🔗External FK Prefix No Longer Prependedv1.3.6
FK references to tables outside the QuickSQL model no longer get the object prefix added. E.g. /fk UC_FLOWFORMS_FORM with prefix IW now correctly generates references UC_FLOWFORMS_FORM.
⚠11 Critical DDL Generation Bugsv1.3.5+
Fixes: native boolean true/false defaults, view FROM spacing, reserved word aliases, trailing AND, duplicate column aliases, wrong FK columns in views, SODA trigger skipping.
🔄ANSI JOIN Syntax in Viewsv1.3.11
Views now use standard LEFT JOIN … ON and CROSS JOIN instead of Oracle-proprietary (+) syntax. Tables are topologically sorted so base tables appear first.
🔒Immutable Table Trigger Correctedv1.3.11
Immutable tables now correctly use BEFORE INSERT only, and the dead ELSIF UPDATING branch for row_version is removed.
